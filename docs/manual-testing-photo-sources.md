# Public Photo Libraries for Manual Testing

Use these sources to build non-sensitive sample sets for ingestion, tagging, and search evaluation.

## Recommended Sources

1. Unsplash

- URL: https://unsplash.com
- Strength: high-quality real-world photography and varied scenes.

2. Pexels

- URL: https://www.pexels.com
- Strength: broad free stock catalog, easy category sampling.

3. Pixabay

- URL: https://pixabay.com
- Strength: mixed media and broad topical coverage.

4. Wikimedia Commons

- URL: https://commons.wikimedia.org
- Strength: historical, educational, and domain-diverse imagery.

5. Open Images Dataset

- URL: https://storage.googleapis.com/openimages/web/index.html
- Strength: very large dataset with object annotations.

6. COCO Dataset

- URL: https://cocodataset.org
- Strength: strong benchmark set for object-centric validation.

## Usage Guidance

- Build test batches by subject profile (people, vehicles, animals, general).
- Include ambiguous images to evaluate false-positive tagging.
- Include near-duplicates to evaluate consistency.
- Track source, license, and attribution obligations for each sample set.

## Licensing Reminder

Always confirm current licensing and attribution requirements before storing or redistributing sample images.
