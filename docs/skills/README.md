# Skills Baseline

This directory captures practical implementation skills for this project.

## Purpose

- Provide technology-specific baseline practices.
- Record implementation learnings after each successful task.
- Build a repeatable playbook for future tasks.

## Update Rule

After every successful implementation task, update relevant file(s) with a short entry containing:

- What worked
- What failed or almost failed
- Reusable pattern
- Test strategy that validated the change
