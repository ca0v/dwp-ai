# Vanilla CSS Grid Skill Baseline

## Baseline

- Use a token-driven grid system with CSS custom properties.
- Design mobile-first, then scale layout at defined breakpoints.
- Support light/dark mode with explicit color tokens.

## Patterns

- Keep spacing and typography on shared scales.
- Use grid areas and utility classes sparingly and intentionally.
- Preserve clear contrast and touch target sizing.
